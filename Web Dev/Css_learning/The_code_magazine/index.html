The Basic Language of the Web: HTML

Posted by Dev Lalit Kumar in Friday, 10 th November 2021

All mordern websites and web applications are build using three fundamental Technologies- HTML, CSS, JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you should learn it.


What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that developers use to structure and describe the content of a webpage. (not a programming language). It is called the skeleton of the web.   Without HTML, the while internet does not exists!

HTML consists of elements that describe the different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at W3 School website


Why should you learn HTML ?

There are countless reasons for leaning fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language.
  • To hand-craft beautiful websites instead of relying on tools like Workpress or Wix.
  • To build Web Appliations.
  • To impress friends.
  • To have fun and gain experience for income.
